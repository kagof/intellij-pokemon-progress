# Security Policy

## Supported Versions

The latest version will always be the supported version.

## Reporting a Vulnerability

If you have found a security vulnerability in the plugin, please report it using
the [GitHub vulnerability report](https://github.com/kagof/intellij-pokemon-progress/security). If that is not possible,
you may also contact the project maintainer by email.

If the issue is severe enough, you may also want to report it to Jetbrains.
